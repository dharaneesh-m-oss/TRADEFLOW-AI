# Sample Documents

Place sample shipping documents (PDFs, images) here for testing.

## Expected Documents

- Sample invoices (PDF/PNG)
- Bills of lading
- Shipping documents

## Note

For hackathon demo, you can use any shipping-related document.
The system will attempt to extract:
- Invoice numbers
- Dates
- Seller/Buyer names
- Items with quantities and prices
- Total amounts
- HS codes
- Country of origin
- Shipping addresses
